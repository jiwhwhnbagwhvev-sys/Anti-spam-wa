# ANTI-SPAM-REAL (Android + Termux)

## APA INI?
ANTI-SPAM-REAL adalah project Termux berbasis Python + ADB
yang berfungsi untuk:
- Membungkam panggilan spam
- Menyenyapkan nomor tak dikenal
- Mengurangi spam WhatsApp Call
- Mencatat percobaan spam sebagai bukti

Project ini TIDAK:
- Menyerang balik
- Merusak jaringan
- Mengklaim akses server operator

Project ini LEGAL & NYATA.

---

## BAGAIMANA CARA KERJANYA?
Android menyediakan fitur internal:
- Do Not Disturb (DND)
- Silence Unknown Callers
- Notification Control

Tool ini:
- Mengaktifkan fitur itu secara otomatis
- Mengelolanya lewat menu Termux
- Memberi kontrol ON / OFF cepat

---

## KENAPA INI EFEKTIF?
Pelaku spam:
